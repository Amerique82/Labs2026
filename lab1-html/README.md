# Lab 1 - Introduction to HTML

## What this lab covers
Introduction to HTML fundamentals and setting up a development environment with VS Code.

## Files

| File | Description |
|------|-------------|
| `index.html` | Basic page structure — `<!DOCTYPE html>`, `<head>`, `<body>`, title and heading |
| `headings.html` | Heading levels h1–h6 with paragraphs |
| `hyperlinks.html` | Internal links and external links with `target="_blank"` |
| `tables.html` | Table with `<thead>`, `<tbody>`, `<th>`, `<td>` |
| `images.html` | `<img>` tag with `src`, `alt`, and `width` attributes |
| `styles.html` | Inline CSS with classes, colours, fonts |
| `forms.html` | Form with `<input>`, `<select>`, `<textarea>`, `<button>` |
| `html5-structure.html` | Semantic elements: `<header>`, `<nav>`, `<main>`, `<article>`, `<aside>`, `<footer>` |

## Key concepts

- Every HTML file starts with `<!DOCTYPE html>` to tell the browser it's HTML5
- `<head>` contains metadata (not visible on screen)
- `<body>` contains everything the user sees
- Attributes go inside the opening tag: `<img src="photo.jpg" alt="description">`
- CSS selects elements by tag (`h1 {}`) or class (`.highlight {}`)

## How to run
Open any `.html` file in your browser — no server needed.
