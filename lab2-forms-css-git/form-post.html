HTML Form - POST Method

The POST method sends form data in the request body — it does NOT appear in the URL. This is more secure for sensitive data like passwords or personal information.

Developer Tools tip: After submitting, open DevTools (F12 or Cmd+Option+I), go to Network tab → click the request → go to Payload. Notice the data is in the body, NOT in the URL — unlike GET.
