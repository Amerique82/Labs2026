HTML Form - GET Method

The GET method sends form data as URL parameters. You can see the values in the browser address bar after submitting. Example: form-get.html?name=Amerique&email=a@test.com

Developer Tools tip: After submitting, open DevTools (F12 or Cmd+Option+I), go to the Network tab, and look at the request. Notice the parameters appear in the URL.
